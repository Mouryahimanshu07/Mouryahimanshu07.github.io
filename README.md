### ⚡ Interactive Features

#### 🤖 Gemini AI Q&A Assistant
> Ask anything about Himanshu's background, projects, and skills.
- Keyword-based knowledge engine trained on resume data
- Suggested quick-question chips
- Real-time chat interface with bot/user message bubbles

#### 🦴 Bone Age Prediction Simulator
> Live simulation of the CNN Transfer Learning model
- Select X-ray scan type: Child (6 Yrs) / Teen (14 Yrs) / Adult (21 Yrs)
- Displays predicted bone age, CNN confidence score & model architecture
- Animated heatmap overlay & inference loading simulation

#### 😊 7-Emotion Classifier Tester
> Real-time facial sentiment evaluation across all 7 emotion states
- All 7 emotions from the resume: **Angry, Disgust, Fear, Happy, Neutral, Sad, Surprise**
- Live probability bars sorted by confidence score
- Animated face bounding box with emotion label

#### 💻 Interactive Developer CLI Terminal (`mourya-cli`)
> A fully functional terminal in the browser

```bash
himanshu@ai:~$ help               # List all available commands
himanshu@ai:~$ skills             # View tech stack & libraries
himanshu@ai:~$ projects           # List all AI/ML projects
himanshu@ai:~$ education          # View CGPA & college details
himanshu@ai:~$ contact            # Show email, phone & social links
himanshu@ai:~$ run gemini_qa.py   # Simulate Gemini QA execution
himanshu@ai:~$ run bone_age.py    # Simulate bone age inference
himanshu@ai:~$ run emotion.py     # Simulate emotion detection
himanshu@ai:~$ clear              # Clear terminal
```

#### 📄 Resume PDF Viewer
